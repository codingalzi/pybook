5. 기본자료형¶

일반적으로 자료형으로 불리는 데이터 유형data type은 사용되는 데이터와 관련된 많은 정보를 포함한다. 해석기interpreter 또는 컴파일러compiler는 자료형에 포함된 정보를 이용하여 해당 데이터의 사용법을 확인한다. 이번 장에서는 파이썬 프로그래밍 언어가 제공하는 기본 자료형 네 가지를 간단히 살펴본다.

  • 정수

  • 부동소수점

  • 부울값

  • 문자열

5.1. 정수¶

일반적으로 알고 있는 -2, -1, 0, 1, 2 등의 정수(자연수, 0, 음의 정수)는 int 자료형의 값이다. 정수형으로 나타내면 덧셈, 뺄셈, 곱셈, 나눗셈 등의 기본 연산이 가능하다. 정수는 사칙연산, 변수 할당 등에 사용된다.

1 + 2

3

a = 4
a * 2

8

type() 함수는 주어진 값의 자료형을 확인해준다.

type(a)

int

5.2. 부동소수점¶

부동소수점은 유리수, 실수 등을 가리키며 float 유형의 값이다. 1.2, 0.333, -1.2, -3.7680, 4.0 등은 모두 부동소수점형(float)이다. 정수와 마찬가지로 사칙연산, 변수 할당 등에 사용된다.

부동소수점은 원래 실수를 컴퓨터에서 다루기 위해 개발되었으나 실제로는 유한 소수만을 다룬다. 예를 들어 무리수인 원주율 \(\pi\)의 경우에도 컴퓨터의 한계로 인해 소수점 이하 적당한 자리에서 끊어서 사용한다.

2.1 + 3.3

5.4

c = 2.1
type(c)

float

정수와의 연산도 가능한데, 이때 정수를 부동소수점으로 취급한다. 예를 들어 3은 3.0으로 처리된다.

2.3 + 3

5.3

2.3 + 3.0

5.3

5.3. 문자열¶

문자열string은 문자 기호로 이루어진 단어, 문장 등을 가리킨다. 키보드에 포함된 영문 알파벳, 한글 자음과 모음 등을 기본적으로 사용한다. 문자열은 세 가지 방식으로 작성될 수 있다.

작은 따옴표 활용

s = '안녕하세요.'
print(s)

안녕하세요.

큰 따옴표 활용

s = "안녕하세요."
print(s)

안녕하세요.

연속된 세 개의 작은 또는 큰 따옴표 활용

여러 줄로 이루어진 문자열 작성에 사용된다.

s = '''안녕.
잘 지내?'''

print(s)

안녕.
잘 지내?

s = """안녕.
잘 지내?"""

print(s)

안녕.
잘 지내?

스페이스와 탭을 활용해서 생성된 여백white space도 문자열로 처리된다.

s = '''안녕.
    잘 지내?'''

print(s)

안녕.
    잘 지내?

s = """안녕.
    잘 지내?"""

print(s)

안녕.
    잘 지내?

유니코드

키보드 상에 존재하지 않는 다른 기호나 문자도 유니코드unicode 형식으로 지원된다. 대표적으로 서유럽 언어에서 많이 사용되는 à, â, ç, è,ü 등 특수 알파벳 등이 유니코드로 지원된다.

또한 이모티콘도 지원된다. 예를 들어 웃는 얼굴의 이모티콘을 나타내는 문자열은 다음과 같다.

>>> s = '\U0001f600'
>>> s
'😀'
>>> s * 2
'😀😀'

유니코드로 지원되는 모든 이모티콘은 Emoji Charts에서 확인할 수 있다. 단, U+로 시작하는 부분을 U000 으로 대체해서 사용해야 한다.

5.3.1. 백슬래시와 이스케이프 문자¶

문자열 자체에 따옴표가 포함되면 조심해야 한다. 예를 들어, 작은 따옴표가 포함된 문자열을 다음과 같이 지정하면 오류가 발생한다.

>>> s = 'Python's grammar'
  File "<stdin>", line 1
    s = 'Python's grammar'
                ^
SyntaxError: invalid syntax

이유는 작은 따옴표로 문자열의 시작을 지정했기 때문에 Python's 문장에 사용된 작은 따옴표가 문자열의 끝을 의미하게 되는데 이후에도 문자열이 이어지게 되어 결국 문자열의 끝이 불분명해져서 구문 오류를 뜻하는 SyntaxError가 발생한다.

이런 종류의 오류를 방지하는 다양한 방식이 존재하지만, 여기서는 백슬래시backslash 기호 (\)를 따옴표 바로 앞에 추가하는 방식만 소개한다.

s = 'Python\'s grammar'
print(s)

Python's grammar

슬래시 기호(\)와 원화 기호(₩)

백슬래시 키는 키보드의 Enter 키 바로 위에 위치하는데 한글 키보드의 경우 원화 기호 키 ₩ 가 대신 자리잡고 있을 수 있다. 그리고 모니터 화면에도 사용하는 운영체제의 언어 설정에 따라 원화 기호(₩)와 백슬래시 기호(\) 둘 중에 하나로만 보인다. 하지만 단순한 언어 설정의 차이일 뿐 기능은 동일하다.

백슬래시는 이처럼 특정 문자와 함께 사용되면 특별한 기능을 갖는 이스케이프 문자escape character를 구성한다. 가장 많이 사용되는 이스케이프 문자의 목록은 다음과 같다.

5.3.3. whitespace¶

whitespace는 화면상으로 아무것도 표시되지 않는 공백 문자를 말한다.

string.whitespace는 공백으로 간주하는 문자를 포함하는 문자열로, 여기에는 스페이스space, 탭tab, 줄변경line feed or new line, 캐리지 리턴carriage return, 수직탭vertical tab, 폼피드form feed 문자가 포함된다.

import string
string.whitespace

' \t\n\r\x0b\x0c'

기호

설명

공백 space

\t

탭 tab

\n

줄변경 line feed or new line

\r

캐리지 리턴 carriage return

\v 또는 \x0b

수직탭 vertical tab

\f 또는 \x0c

폼피드form feed

 
print('1\n2\t3')

1
2	3

print('123\b4')

1234

  • 캐리지 리턴(\r)은 그 줄의 맨앞으로 이동시키는 특수 문자이다.

print('1234\r56')

1234
56

  • \v 또는 \x0b는 수직탭이고, \f 또는 \x0c는 다음 페이지로 이동하는 것을 의미한다.

문자열 ''과 문자의 ' ' 차이첨

빈 문자열empty string ''는 아무런 문자도 포함하지 않았기 때문에 문자열의 길이 즉, 문자열에 포함된 문자의 개수가 0이다.
반면 ' '는 눈에 보이지는 않지만 공백 문제 하나를 포함하는 문자열이며 길이가 1이다.

5.3.4. 문자열에 특수 문자 활용하기¶

백슬래시(￦), 줄바꾸기(￦n), 탭(￦t) 등은 문자열에 사용될 경우 특수한 기능을 갖는다. 그리고 백슬래시(\)기호를 붙여서 특수 기능을 해제escape할 수 있다.

print("Hello\\n World")

Hello\n World

print("Hello\\t World")

Hello\t World

print("Good\\night")

Good\night

연속된 백슬래시 두 개를 출력하려면 아래와 같이 해야 한다.

print("\\\\")

\\

주의

아래와 같이 코드를 작성하면 오류가 발생하므로 주의해야 한다.

>>> print("\\\")
  File "/tmp/ipykernel_76/2337434698.py", line 1
    print("\\\")
                ^
SyntaxError: EOL while scanning string literal

참고

순수 문자열 ‘가공되지 않은’의 의미를 갖는 ‘raw’ 단어의 첫 글자인 ‘r’을 문자열 앞에 두면 특수 기능이 사라진다.

print(r"Hello\n World")

Hello\n World

print(r"Hello\t World")

Hello\t World

print(r"Hello\ World")

Hello\ World

5.3.6. 문자열과 print() 함수¶

그 동안 출력을 위해 print() 함수를 사용하였다. 예를 들어, Hello, python을 출력하고 싶다면, 아래와 같이 코드를 작성하면 된다.

print('Hello, python')

Hello, python

이제 print() 함수에 대해서 조금 더 살펴보자.

  • 큰 따옴표(또는 작은 따옴표)로 둘러싸인 문자열을 연속해서 사용하면 문자열에서 +연산을 한 것과 동일한 결과를 출력해준다.

print("Hello ""World")
print('Hello ''World')
print('Hello '"World")
print("Hello " + "World")

Hello World
Hello World
Hello World
Hello World

  • 문자열 사이에 콤마(,)를 사용하면 띄어쓰기를 할 수 있다.

print("Hello", "World")

Hello World

  • 인자들 사이에 구분자를 넣고 싶다면, sep을 변경하면 된다. sep의 기본값은 공백이다.

print("010", "1234", "5678", sep = "-")
print("Hello", "World", sep = ", ")

010-1234-5678
Hello, World

  • 마지막에 출력할 문자를 변경하고 싶다면, end를 변경하면 된다. end의 기본값은 줄변경(\n)이다.

print("a")
print("b", end = "")
print("c", end = "\t")
print("d")

a
bc	d

5.3.7. 문자열 포매팅 String formatting¶

5.3.7.1. str.format()¶

format()메서드를 사용하면, 문자열의 중괄호({})가 format()의 인자로 변경된다. 예제와 함께 살펴보자.

'{}님, 안녕하세요.'.format('강현')

'강현님, 안녕하세요.'

name = '강현'
'{}님, 안녕하세요.'.format(name)

'강현님, 안녕하세요.'

age = 3
'강현이는 {}살이다.'.format(age)

'강현이는 3살이다.'

여러 개의 값을 변경할 때는 콤마(,)로 구분해서 적어준다.

name = '강현'
age = 3
'{}이는 {}살이다.'.format(name, age)

'강현이는 3살이다.'

인덱스 항목을 사용하여 넣어줄 위치를 지정할 수도 있다.

name1 = '강현'
name2 = '나영'
print('{}이와 {}이는 친구다.'.format(name1, name2))
print('{0}이와 {1}이는 친구다.'.format(name1, name2))
print('{1}이와 {0}이는 친구다.'.format(name1, name2))

강현이와 나영이는 친구다.
강현이와 나영이는 친구다.
나영이와 강현이는 친구다.

소수점 표현

콜론(:) 뒤에 소수점아래 몇 번째 자리까지 출력할지를 적어주면, 그 만큼을 보여준다.
예제와 함께 살펴보자.

  • .은 소수점을 의미하고, 소수점 뒤의 숫자는 소수점 뒤에 나올 숫자의 개수다.

num = 0.123456789
print('{0:.1f}'.format(num))
print('{0:.2f}'.format(num))
print('{0:.3f}'.format(num))
print('{0:.5f}'.format(num)) # 소수점 아래 여섯 번째 자리에서 반올림

0.1
0.12
0.123
0.12346

5.3.7.2. f-string¶

문자열 앞에 f 를 붙이면, 문자열 포매팅 기능을 사용할 수 있다.
예제와 함께 살펴보자.

name = '강현'
f'{name}님, 안녕하세요.'

'강현님, 안녕하세요.'

age = 3
f'강현이는 {age}살이다.'

'강현이는 3살이다.'

name = '강현'
age = 3
f'{name}이는 {age}살이다.'

'강현이는 3살이다.'

{}안에 변수와 수식(+, -, *, / 등)을 함께 사용하는 것도 가능하다.

name = '강현'
age = 3
f'{name}이의 동생은 {age - 2}살이다'

'강현이의 동생은 1살이다'

num = 0.123456789
print(f'{num:.1f}')
print(f'{num:.2f}')
print(f'{num:.3f}')

0.1
0.12
0.123

5.4. 부울값¶

참(True)과 거짓(False)를 나타내는 자료형이다. 파이썬에서 부울 자료형은 int 형의 자식형 subtype이고, 대부분의 상황에서 True와 False는 각각 1과 0처럼 동작한다.

5.5. 형변환¶

2.3 + 3을 실행하면 파이썬 해석기interpreter는 내부적으로 정수를 부동소수점으로 형변환하는 함수 float()를 이용하여 정수 3을 부동소수점 3.0으로 유형을 변환한 후에 덧셈을 처리한다. 이렇게 값의 유형을 적절하게 자동 변환하는 기능을 형변환type casting이라 부른다.

float(3)

3.0

float() 함수 이외에 int(), str(), bool() 등 다양한 형변환 함수가 제공된다.

int() 함수

정수 모양의 문자열 또는 부동소수점을 정수로 변환한다.

int('5')

5

int(4.8)

4

True와 False는 각각 1과 0으로 취급된다.

int(True)

1

float() 함수

유한소수 모양의 문자열 또는 정수 등을 부동소수점으로 변환한다.

float('7.9')

7.9

str() 함수

임의의 값을 문자열로 변환한다.

str(6)

'6'

str(7.6)

'7.6'

str(False)

'False'

bool() 함수

임의의 값을 True 또는 False 로 변환한다. 0, 0.0, ''(빈 문자열) 등처럼 0, 비어 있는 것, 의미 없는 것등은 False 로 그렇지 않으면 True 로 지정한다.

bool(0)

False

bool(2)

True

bool(0.0)

False

bool(0.01)

True

bool('')

False

bool('Hello')

True

주의

int()함수의 인자로 문자열을 사용할 때는 그 모양이 정수모양이어야 하고, float() 함수의 인자로 문자열을 사용할 때는 그 모양이 정수 또는 부동소수점 모양이어야 한다.
