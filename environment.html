2. 프로그래밍 개발 환경#

파이썬으로 프로그래밍을 하려면 적절한 개발환경을 준비해야 한다. 파이썬 프로그래밍 개발 환경은 크게 오프라인 환경과 온라인 환경으로 나뉜다.

  • 온라인 개발 환경: 인터넷 브라우저를 이용하여 특정 서버에 접속해서 편집기와 파이썬 실행기를 활용하는 환경

  • 오프라인 개발 환경: 편집기와 파이썬 실행기를 PC에 직접 설치해서 활용하는 환경

프로그래밍이 처음이라면 먼저 온라인 방식을 사용해서 파이썬과 어느 정도 친숙해진 다음에 오프라인 방식을 시도할 것을 추천한다.

2.1. 온라인 개발 환경: 구글 코랩#

구글 코랩Google Colab은 구글에서 제공하는 파이썬 전용 온라인 주피터 노트북이다. 웹브라우저를 이용하여 어떤 준비 없이 바로 파이썬 프로그래밍을 시작할 수 있다.

구글 코랩의 간단한 사용법은 유튜버 봉수골 개발자 이선비의 동영상을 참고한다. 그리고 입문용은 아니지만 구글 코랩만이 지원하는 유용한 고급 기능을 TensorFlow 팀의 동영상에서 확인할 수 있다.

구글 코랩을 사용하다 보면 특정 라이브러리를 설치해야하는 경우가 발생할 수 있는데 그럴 때는 pip 파이썬 라이브러리 관리자를 이용하여 설치하면 된다. 예를 들어 beautifulsoup4 라이브러리를 추가 설치하려면 아래 명령문을 주피터 노트북 코드셀에서 실행한다.

!pip install beautifulsoup4

pip은 파이썬 명령어가 아니라 컴퓨터 운영체제 관리에 사용되는 쉘shell 명령어이다. 이런 쉘 명령어를 터미널이 아닌 주피터 노트북 셀에서 실행하려면 느낌표(!)를 함께 사용한다.

쉘과 터미널

쉘shell은 컴퓨터 운영체제의 기능과 서비스를 관리하거나 구현하는 데에 사용되는 명령문command lines을 실행하는 실행기interpreter이다. 즉, 사용자와 컴퓨터 커널kernel을 연결하는 다리 역할을 수행한다. 쉘이 실행하는 명령문을 쉘 스크립트shell script라 부른다. 다양한 종류의 쉘이 존재하지만, 리눅스의 기본 쉘인 bash(배시)와 맥 OSX의 zsh(Z 쉘)이 가장 많이 사용된다.

반면에 터미널terminal은 쉘 스크립트를 작성하고 실행시킬 수 있는 사용자 인터페이스user interface(UI)이며 콘솔console이라고도 불린다. 윈도우 사용자의 경우 아나콘다 패키지와 함께 설치된 Anaconda Prompt (anaconda3) 터미널을 이용할 수 있다.

2.2. 오프라인 개발 환경: 아나콘다#

인터넷 연결 문제, 보안 등의 이유로 인터넷이 단절된 상황에서의 개발 환경이며 개발자가 직접 자신의 컴퓨터에 설정한다.

2.2.1. 파이썬 설치#

파이썬 개발환경 설치가 처음이라면 데이터 분석 관련 중요 라이브러리, 스파이더 편집기, 주피터 노트북을 모두 포함하는 아나콘다Anaconda 파이썬 패키지를 추천한다. 자세한 설치 요령은 아나콘다 설치를 참고한다.

2.2.2. 주피터 노트북 실행#

Anaconda Prompt (anaconda3) 등 임의의 터미널에서 아래 명령을 차례대로 실행하면 주피터 노트북Jupyter notebook을 열거나 새로 작성할 수 있다.

(base) $ jupyter notebook

윈도우 사용자는 아래 명령도 사용할 수 있다.

(base) $ jupyter-notebook.exe

인터넷 브라우저에서 아래와 같이 주피터 서버가 실행되면 원하는 폴더에서 기존의 노트북을 실행, 수정하거나 새로운 주피터 노트북을 생성할 수 있다.

주피터 노트북 사용법

주피터 노트북 실행법에서 간단한 사용법을 확인할 수 있다. 아나콘다의 설치 과정과 주피터 노트북의 기초 사용법에 대한 보다 상세한 설명은 유튜버 나도코딩의 동영상을 참고하면 좋다.

2.3. 오프라인 통합 개발 환경(IDE)#

대형 프로젝트를 진행하면 많은 코드 파일을 작성하고 관리해야 한다. 이때 코드 편집과 실행, 디버깅, 파일 관리 등 프로그래밍 관련 모든 작업을 통합해서 지원하는 통합 개발 환경을 사용한다. 일명 IDEInteractive Development Environment라고 불리는 통합 개발 환경을 지원하는 대표적인 상용 앱과 무료 앱은 다음과 같다.

  • 파이참Pycharm: 가장 인기있는 상용 프로그램. Pro 와 Community 두 버전을 지원하며 입문자는 무료 버전인 Community 추천.

  • 비주얼 스튜디오 코드Visual Studio Code: 마이크로소프트에서 지원하며, 현재 가장 인기 있는 무료 프로그램. 파이썬 뿐만 아니라 거의 모든 프로그래밍 언어를 위한 통합 개발 환경 지원.

여기서는 일명 VS Code라 불리는 비주얼 스튜디오 코드를 사용할 것을 추천한다. VS Code의 설치 과정과 사용법은 비주얼 스튜디오 코드 설치와 생활코딩의 동영상을 참고한다.
