코드 셀#

코드 셀에서는 파이썬 코드를 작성하고 실행할 수 있다.

print("숫자맞히기 게임에 환영합니다.")

secret = '3'
guess = input("0부터 9 사이의 숫자 하나를 입력하세요: ")

if guess == secret:
    print("맞았습니다!")
else:
    print("틀렸습니다!")

print("게임 종료!")
